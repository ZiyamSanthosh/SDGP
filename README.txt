
/************************* FLASK API *************************/

FlaskAPI - contains the machine learning model for prediction

To start testing:
	run FlaskAPI with 'python app.py'

Requirements:
	flask
	keras
	tensorflow
	flask_cors
	numpy
	joblib
	
Set up python environment:
	1. Install tensorflow, keras, flask from anaconda navigator base environment.  
	2. Run 'pip install flask_cors' in the pycharm terminal
	3. Execute 'python app.py' in terminal to run the flask app



/************************ NODE API *************************/

NodeAPI - contains the backend for the mobile application 

To run:
	1. run command 'npm install' to install all the required modules
	2. run command 'nodemon'or 'npm start' in cases nodemon is not working
